Class ProfilePreview

This class provides a control that gives you a sprite view of an entity with an applied HumanoidCharacterProfile, and some buttons to rotate the view. It's a relatively lightweight wrapper around ProfilePreviewSpriteView specifically for the right side panel of the HumanoidProfileEditor window.
