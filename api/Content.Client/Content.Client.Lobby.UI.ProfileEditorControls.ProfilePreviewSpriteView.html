Class ProfilePreviewSpriteView

This class provides a control that gives you a sprite view of an entity with an applied HumanoidCharacterProfile. It handles the loading and spawning of a profile and also extracts information about the profile such as the profile's name, loadout override name, and preferred job if available.

LoadPreview(ICharacterProfile, JobPrototype, Boolean)

Create an entity from a character profile and display it in the sprite view. This can be used to reload the profile preview when job clothes change, but shouldn't be used for things on a slider like skin color. Use ReloadProfilePreview(ICharacterProfile) for that.
