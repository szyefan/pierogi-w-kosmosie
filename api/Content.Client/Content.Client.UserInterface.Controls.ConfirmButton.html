Class ConfirmButton

A Button that requires a second click to actually invoke its OnPressed action.
When clicked once it will change rendering modes to be prefixed by ConfirmPrefix and displays ConfirmationText on the button instead of Text.

After the first click CooldownTime needs to elapse before it can be clicked again to confirm.
When the button doesn't get clicked a second time before ResetTime passes it changes back to its normal state.
