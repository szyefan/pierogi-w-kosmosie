GetRulesForFont(Nullable<String>, NotoFontFamilyStack, List<(Nullable<String>, Int32)>)

Get the style rules for the given font stack, with the provided sizes. Does not set the 'default' font, but does create rules for every combination of font kind and font size. This is intended for font sizes you think will be common/generally useful, for less common usecases prefer specifying the font explicitly.
