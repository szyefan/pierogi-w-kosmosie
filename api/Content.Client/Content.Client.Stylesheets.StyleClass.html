Class StyleClass

A collection of public reusable style classes. These should be general purpose (Not specific to only one element or Ui).

Inheritance
System.Object
StyleClass
Namespace: Content.Client.Stylesheets
Assembly: Content.Client.dll
Syntax
public static class StyleClass : object
Remarks

It is named StyleClass as opposed to StyleClasses because StyleClasses is a field on Control so it made it a pain to reference this class from a Control. (Weird name is worth typing StyleClass.OpenBoth vs. Stylesheets.Styleclasses.OpenBoth)
