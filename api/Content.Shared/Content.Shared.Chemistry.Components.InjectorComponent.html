Class InjectorComponent

Implements draw/inject behavior for droppers and syringes.

Inheritance
System.Object
InjectorComponent
Namespace: Content.Shared.Chemistry.Components
Assembly: Content.Shared.dll
Syntax
public sealed class InjectorComponent : Component
Remarks

Can optionally support both injection and drawing or just injection. Can inject/draw reagents from solution containers, and can directly inject into a mobs bloodstream.
