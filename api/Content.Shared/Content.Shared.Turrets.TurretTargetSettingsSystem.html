Class TurretTargetSettingsSystem

This system is used for validating potential targets for NPCs with a TurretTargetSettingsComponent (i.e., turrets). A turret will consider an entity a valid target if the entity does not possess any access tags which appear on the turret's ExemptAccessLevels list.
