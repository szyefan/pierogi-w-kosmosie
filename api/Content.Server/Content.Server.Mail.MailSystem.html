Parameters
Type Name Description
EntityUid uid
MailComponent component
System.String localizationString
Remarks

This will mark a parcel as no longer being profitable, which will prevent multiple failures on different conditions for the same delivery.

The standard penalization is breaking the anti-tamper lock, but this allows a delivery to fail for other reasons too while having a generic function to handle different messages.
