Class StoryBeatPrototype

A point in the story of the station where the dynamic system tries to achieve a certain level of chaos for instance you want a battle (goal has lots of hostiles) then the next beat you might want a restoration of peace (goal has a balanced combat score) then you might want to have the station heal up (goal has low medical, atmos and power scores)

In each case you create a beat and string them together into a story.

EndIfAnyWorse might be used for a battle to trigger when the chaos has become high enough. endIfAllBetter is suitable for when you want the station to reach a given level of peace before you subject them to the next round of chaos.
