Struct DatabaseNotification

Represents a notification sent between servers via the database layer.

Namespace: Content.Server.Database
Assembly: Content.Server.dll
Syntax
public struct DatabaseNotification
Remarks

Database notifications are a simple system to broadcast messages to an entire server group backed by the same database. For example, this is used to notify all servers of new ban records.

They are currently implemented by the PostgreSQL NOTIFY and LISTEN commands.
